# Filtro de Leads

Esta es una aplicación simple para filtrar leads desde un archivo de Excel y descargar los resultados como un archivo CSV.

## Instrucciones de Uso

1. **Instalar dependencias**:
2. 2. **Iniciar el servidor**:

3. **Abrir la aplicación en el navegador**:
Visita `http://localhost:4000` en tu navegador web.

4. **Introducir los criterios de filtro**:
Completa los campos `Role`, `Industry`, `Country`, y `CNAE` con los criterios deseados y haz clic en "Filtrar y Descargar".

5. **Descargar el archivo CSV**:
Los datos filtrados se descargarán automáticamente como un archivo CSV.

## Estructura del Proyecto

- `public/`: Contiene los archivos estáticos (HTML, CSS, JavaScript).
- `data/`: Contiene el archivo de Excel `ALL_LEADS.xlsx`.
- `server.js`: Archivo principal del servidor Node.js.
- `package.json`: Archivo de configuración de npm.
- `README.md`: Este archivo.
